Projeto: GitHub User Activity

No meio do ano de 2025, consegui meu primeiro estágio como desenvolvedor backend de Php/Laravel pela SEMAN na SEMIT.
Não possuia muita experiência, mas dediquei-me bastante para aprender mais tanto a linguagem tanto o framework,
assistir vários tutoriais no youtube, tive auxílio de colegas de trabalho, onde dividiram conhecimento e algumas
plataformas para meu aprendizado.

Claro que não parei por aí. O desenvolvimento backend possui outras camadas além de saber sobre o léxico de Php
e Laravel. Para tanto, busquei no site roadmap.sh o contéudo sobre backend,
a fim de, tornar-me um desenvolvedor melhor. Estudei a parte inicial e meu primeiro projeto é um programa que
interage com o usuário a partir da Linha de Comando (CLI), daí, preparando o arquivo "enviroment.php", que deve
possuir seu Toke ou API Token, você consegue rastrear a atividade de um usuário qualquer.

O link do meu repositório é: GitHub User Activity

Tecnologias utilizadas: Php, API Rest.
